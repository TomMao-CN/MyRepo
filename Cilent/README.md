### 安装的组件
```
【font-awesome】
font-awesome字体图标库

import 'font-awesome/scss/font-awesome.scss'
```
```
【axios】

npm install --save axios vue-axios

import axios from 'axios'
import VueAxios from 'vue-axios'
Vue.use(VueAxios, axios)
```
